# Minimal Academic Homepage

This is a very small Jekyll-based academic homepage designed for GitHub Pages.

## Structure

- `_config.yml`: site-level configuration
- `index.md`: home page
- `publications.md`: publications page
- `projects.md`: projects page
- `cv.md`: CV page
- `_layouts/default.html`: shared HTML layout
- `assets/style.css`: site styling

## Deploy

1. Upload all files to the root of your repository `Xian-Bei.github.io`.
2. Commit and push to the `main` branch.
3. In GitHub: `Settings -> Pages`, ensure the site is deployed from the main branch root.
4. Wait for GitHub Pages to build.

## Personalization checklist

- Replace placeholder email in `_config.yml` and `index.md`
- Add your Google Scholar link in `index.md`
- Add `assets/cv.pdf`
- Update publication titles in `publications.md`
